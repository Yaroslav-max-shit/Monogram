# Monogram Messenger - Инструкция по установке и запуску

Добро пожаловать Monogram Messenger — современный, безопасный и удобный мессенджер.

## 🔧 Установка

Установка была автоматически выполнена через `setup.py`. Все необходимые зависимости установлены.

## ▶️ Запуск приложения

### Способ 1: Через графический интерфейс (рекомендуется)
1. Откройте файл `Monogram.html` двойным щелчком.
2. Нажмите кнопку **«Запустить Monogram»**.
3. Приложение запустится автоматически, и браузер откроет интерфейс.

### Способ 2: Через файл Python
1. Найдите файл `monogram.py` в папке с программой.
2. Дважды щелкните по нему.
3. Приложение запустится.

## ⏹️ Остановка приложения

Для остановки:
- Нажмите `Ctrl+C` в консоли, где запущено приложение.

## 🌐 Как работает

- `monogram.py` запускает сервер (бэкенд на FastAPI) и фронтенд (на Vite).
- Фронтенд доступен по адресу: `http://localhost:5173`.
- Все данные хранятся локально.

## 📄 Полная документация

Для получения полной инструкции запустите файл `Monogram.html`.
